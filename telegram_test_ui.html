Управление Webhook

Важно: Для работы Webhook ваш скрипт должен быть развернут как веб-приложение. Нажмите "Развертывание" -> "Новое развертывание" в редакторе Apps Script, выберите тип "Веб-приложение" и скопируйте ID развертывания в поле выше.
